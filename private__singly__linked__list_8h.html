Detailed Description

The Private Singly Linked List Types and Interface.

The private singly linked list interface and type management. The singly linked list aims to be a simple and efficient push to and pop from front alternative to the doubly linked list. Notably, it can be sorted with the same space and runtime guarantees as the doubly linked list variant with some additional implementation efforts.
